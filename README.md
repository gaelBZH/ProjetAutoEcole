# Mini-Projet Auto École
Ce projet en C# vise à numériser la gestion d'une auto-école en remplaçant les processus manuels par une application en localhost. L'objectif est d'automatiser le suivi des élèves, des moniteurs et des véhicules. L'architecture logicielle s'appuie sur une structure de données MySQL ou PostgreSQL. Le système repose sur des entités clés comme Client, Moniteur ou Leçon.
